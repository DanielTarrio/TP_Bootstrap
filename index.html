Descripción y Características

Es un sistema de control de stock desarrollado para ser accesible desde cualquier lugar, sencillo de usar, escalable. Basado en tecnología web, no necesita de instalación de clientes, facilitando su mantenimiento.

El sistema es ampliamente configurable, permitiendo crear perfiles de usuario a medida con las restricciones que se desee.

Puede manejarce con claves propias o utilizar la seguridad del dominio según la necesidad del cliente

Maneja:

  • Catalogo de materiales
  • Ubicaciones
  • Personal
  • Centros de costo
  • Rutas
  • Costo Promedio Ponderado
  • Ultimo Costo
  • Solicitudes y Consumos
  • Alarmas de stock mínimo y reposición

Tecnologias utilizadas

El HTML5, CSS y Js (HyperText Markup Language, Cascade style shets y Javascritp) la base de la programacion Web

PHP (Hypertext Preprocessor) es un lenguaje de código abierto adecuado para el desarrollo web

jQuery es una biblioteca de JavaScript que se utiliza para simplificar la creación de páginas web dinámicas

CodeIgniter es un framework web de código abierto para PHP.

SqlServer: Poderosa base de datos SQL de Microsoft, que permite almacenar toda la información
